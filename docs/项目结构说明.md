# 项目结构说明

## 目录职责

`android-app/`

- Android 客户端工程目录
- 后续承载短剧列表、播放页、高光互动、尾集分支等客户端逻辑

`server/`

- 本地服务端工程目录
- 后续承载内容管理、高光下发、互动统计、分支任务、AI 编排等能力

`admin-web/`

- 本地 Web 管理后台目录
- 后续承载高光管理、资源配置、互动数据查看、任务管理、演示数据初始化

`scripts/`

- 离线处理脚本目录
- 包含高光提取、文本结构化、分支生成辅助脚本等

`data/`

- 本地数据库、种子数据、导出数据目录

`assets/`

- 结构化静态资源目录
- 包含封面、预生成分支资源、特效资源等

`videos/`

- 本地原始视频资源目录
- 不进入 Git

`config/`

- 配置模板目录

## 当前说明

当前为骨架初始化阶段，目录已经按技术方案完成拆分，后续可以直接基于此结构继续生成各模块工程。
